# <span style="color:#FFA500;"> Формулы которые используются в функциях
### <span style="color: #FF4500;"> Формулы для нахождения площади фигур (для функции area)
- **Круг: S = πR²**
- **Квадрат: S = a²**

### <span style="color: #FF4500;"> Формулы для нахождения периметра фигур (для функции perimeter)
- **Круг: P = 2πR**
- **Квадрат: P = 4a**
